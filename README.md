## Level 1 Tasks

### Level 1 Task 1: Palindrome Checker

**File**: `L1task2.java`

- **Description**: Checks whether a given word or phrase is a palindrome (reads same forwards and backwards).
- **Key Skills**: String manipulation, loops.

### Level 1 Task 2: Student Grade Calculator

**File**: `L1task3.java`

- **Description**: Calculates the average grade of a student based on input marks and assigns a grade (A-F).
- **Key Skills**: Arrays, arithmetic operations.

### Level 1 Task 3: Password Generator

**File**: `L1task4.java`

- **Description**: Generates a random password with customizable length and character sets (upper, lower, numbers, special).
- **Key Skills**: Random number generation, string manipulation.

---

## Level 2 Tasks

### Level 2 Task 1: Password Strength Checker

**File**: `L2task1.java`

- **Description**: Analyzes a password and determines its strength based on length, case, numbers, and special characters.
- **Key Skills**: String analysis, conditional logic.

### Level 2 Task 2: File Encryption/Decryption

**File**: `L2task2.java`

- **Description**: Encrypts or decrypts a text file using a simple Caesar Cipher algorithm.
- **Key Skills**: File I/O, character shifting.

### Level 2 Task 3: Chat Application

**File**: `L2task3.java`

- **Description**: A console-based chat application supporting both Server and Client modes using Java Sockets.
- **Key Skills**: Networking (Sockets), Multi-threading.

## How to Run

Ensure you have **Java Development Kit (JDK)** installed.

1. **Compile all files**:

   ```bash
   javac L1task2.java L1task3.java L1task4.java L2task1.java L2task2.java L2task3.java
   ```

2. **Run a specific task**:
   - Palindrome Checker: `java L1task2`
   - Grade Calculator: `java L1task3`
   - Password Generator: `java L1task4`
   - Password Strength: `java L2task1`
   - File Encrypt/Decrypt: `java L2task2`
   - Chat App: `java L2task3` (Run twice in separate terminals: choose 1 for Server, 2 for Client).
